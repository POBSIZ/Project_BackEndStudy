📖How To Install Ubuntu & Django📖
  • 0. Ubuntu iso 다운로드
    Ubuntu 공식 사이트 바로가기 링크로 들어간후 iso 파일 다운
  • 1. VMware 설치
    이 VMware 사이트 바로가기 링크로 들어간후 각자 OS에 맞게 다운후 설치
  • ctrl+n 눌러 창을 띄우고 전에 다운 해둔 iso 파일 경로 지정
  • Full name, user name, password 등을 자신의 기호대로 설정
  • 이름을 자신의 기호대로 설정후 가상환경 설치 경로 지정
  • 원하는 만큼 용량 지정
  • Customize Hardware 클릭
  • 위의 화면과 동일하게 하드웨어 세팅 (나머지는 삭제)
  • 세팅을 끝내고 Finish 클릭후 우분투가 설치 되기까지 기다리기
  • 위 사진과 같이 나온다면 설치 성공
  • Skip 클릭
  • No 선택
  • 위의 Settings 클릭하여 각자 취향에 맞춰 디스플레이 크기 및 설정 바꾸기 (선택사항)
  • 2. Visual Studio Code 설치
    왼쪽 탭의 파이어폭스 브라우저 접속후 Visual Studio Code 검색
  • 공식 사이트 접속후 .deb 파일 다운
  • 설치후 폴더에 들어가 Downloads 폴더에 들어간다
  • 다운한 파일 실행
  • Install 클릭
  • 왼쪽 아래 어플 탭에 들어가서 설치 되있는지 확인
  • VScode에 들어간후 Extensions 설치
    - Django 1.3.0
    - Django Template
    - Django Snippets
  • - python
  • 3. Python, Django 설치 및 세팅
    ctrl + alt + t 를 눌러서 터미널을 연 후
    sudo apt install python3 입력
    비밀번호 입력
  • mkdir venvs 를 입력하여 venvs 폴더 생성
    cd venvs 만든 폴더로 이동
    sudo apt install python3-venv 입력
    비밀번호 입력
  • python3 -m venv "원하는 venv 명" 입력
  • home/venvs/"위에서 지정한 venv 명"/bin 으로 폴더를 열어서 우클릭하여
    Open In Terminal 클릭
  • source activate 입력하여 가상 파이썬 환경 활성화
    입력구간 맨 앞쪽에 자신이 지정한 venv 명이 괄호 쳐져 나와지면 성공
    이 이후의 과정을 따라하기 위해선 계속 활성화가 유지 되어야 함
  • pip install django==3.1.3 입력하여 장고 설치
  • 오류가 났을시 python3 pip install --upgrade pip 입력 후
    위 명령어 재입력
  • 4. 장고 프로젝트 파일 생성 및 세팅
    cd 를 입력하여 최상위 폴더로 이동
    mkdir projects 을 입력하여 프로젝트 폴더 생성
    cd projects 를 입력하여 프로젝트 폴더로 이동
    mkdir "원하는 프로젝트 파일명" 을 입력하여 폴더 생성
    cd "입력한 프로젝트 파일명" 을 입력하여 폴더로 이동
    django-admin startproject config . 을 입력하여 기본 프로젝트 파일 구성 생성
  • 파일을 생성한 경로에 위와 같이 파일이 생성 되있으면 성공
  • python3 manage.py runserver 를 입력하여 서버를 실행
  • 명령어 입력후 나오는 아이피 주소로 접속
    (지금 당장은 빨간색으로 뜬 오류를 무시해도 된다)
  • 위와 같이 접속이 된다면 장고 설치가 완료 되었습니다.
